选项 A：立即重新编译（推荐）
─────────────────────────────────────────────────────────

1. 安装 .NET 8.0 SDK
   • 访问: https://dotnet.microsoft.com/download/dotnet/8.0
   • 下载并安装 Windows x64 版本
   • 或查看 "手动安装指南.txt"

2. 重新打开 PowerShell（刷新环境变量）

3. 验证安装
   dotnet --version
   # 应显示 8.0.x

4. 构建项目
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
   .\build-with-proxy.ps1

5. 测试新版本
   cd release
   .\Netch.exe


选项 B：先使用现有版本
─────────────────────────────────────────────────────────

1. 使用现有的编译版本
   cd ..\Netch
   .\Netch.exe

2. 稍后再安装 .NET 8.0 SDK 并重新编译


选项 C：仅更新 C# 部分（高级）
─────────────────────────────────────────────────────────

如果只想更新主程序，不重新编译 C++ 组件：

1. 安装 .NET 8.0 SDK

2. 只编译 C# 项目
   dotnet publish Netch\Netch.csproj -c Release -r win-x64 
   --self-contained true -p:PublishSingleFile=true -o release_new

3. 复制旧的 C++ 组件
   Copy-Item ..\Netch\bin\*.dll release_new\bin\
   Copy-Item ..\Netch\bin\*.bin release_new\bin\
   Copy-Item ..\Netch\bin\*.sys release_new\bin\
